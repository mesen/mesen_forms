# MesenForms



## Installation

Add this line to your application's Gemfile:

    gem 'mesen_forms'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mesen_forms


Usage Instruction:
  form_for @site do |f|

Using mesen_forms:
  mesen_form_for @site do |f|
